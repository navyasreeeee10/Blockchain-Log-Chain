# Blockchain Log Chain

## Objective
Create a simple blockchain log chain in Python.

## Features
- Stores logs in blocks.
- Each block contains:
  - Log data
  - Previous block hash
  - Current block hash
- Uses SHA-256 hashing.
- Demonstrates blockchain immutability.

## How It Works
1. Create blocks containing logs.
2. Link each block using hashes.
3. Modify an old block.
4. Observe that the chain becomes invalid.

## Run

python blockchain.py
